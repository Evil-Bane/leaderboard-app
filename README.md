# 🏆 Leaderboard App

[![Live Demo](https://img.shields.io/badge/demo-Live%20Link-success?style=for-the-badge)](https://leaderboard-app-cyan.vercel.app)

A sleek, dynamic leaderboard application built in just a few hours during a hackathon! This project focuses on quick delivery, a visually appealing UI (featuring a video background and splash screen), and pure vanilla web technologies.

## 🚀 Live Demo
Check out the live deployment on Vercel: **[Leaderboard App Demo](https://leaderboard-app-cyan.vercel.app)**

## ✨ Features
* **Built for Speed:** Designed, developed, and deployed within hours at a hackathon.
* **Immersive UI:** Utilizes a video background (`bg.mp4`) for a dynamic, modern feel.
* **Splash Screen:** Engaging entry point powered by custom JavaScript animations (`splash.js`).
* **Frame-based Rendering:** Custom frame handling (`frames.js`) for smooth visual transitions.
* **Zero Dependencies:** Built entirely with vanilla HTML, CSS, and JavaScript. No heavy frameworks, just raw code.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure.
* **CSS3:** Custom styling and responsive design.
* **Vanilla JavaScript:** Logic, state management, and DOM manipulation.
